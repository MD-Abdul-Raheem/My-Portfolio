# Code of Conduct

## Our Pledge

We are committed to maintaining a respectful, professional, and welcoming project environment.

## Expected Behavior

- Be respectful and constructive.
- Give feedback that improves clarity, accuracy, accessibility, or maintainability.
- Respect that this is a personal portfolio and should represent the owner truthfully.
- Use inclusive language.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Adding misleading claims, fake achievements, fake employment, or fabricated credentials.
- Publishing private information without permission.
- Spam or unrelated promotional content.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that do not align with this Code of Conduct.
