# Contributing

Thank you for your interest in improving this portfolio project.

This repository represents a personal portfolio, so contributions should preserve accuracy, accessibility, and professional presentation.

## Development Setup

```bash
npm install
npm run dev
```

## Before Submitting Changes

Please run:

```bash
npm run lint
npm run build
```

## Contribution Guidelines

- Keep content truthful and evidence-based.
- Do not add fake employment, client work, metrics, or credential links.
- Keep Virtual Experience Programs clearly labeled as learning programs, not jobs or internships.
- Preserve responsive behavior across desktop, tablet, and mobile.
- Prefer reusable components and data-driven content updates.
- Do not commit local logs, build output, environment files, or dependency folders.

## Commit Style

Use short, descriptive commit messages, for example:

```text
Improve project card responsiveness
Add certification proof assets
Update portfolio contact details
```

## Reporting Issues

When reporting an issue, include:

- Page or route affected.
- Browser and screen size.
- What you expected.
- What happened instead.
- Screenshot if the issue is visual.
